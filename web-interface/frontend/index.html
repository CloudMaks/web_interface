О проекте

Учебный проект

Веб-интерфейс для учебного киберполигона в рамках выпускной квалификационной работы по специальности 10.05.02.

Цель

Создание платформы для анализа и обработки инцидентов информационной безопасности для студентов и преподавателей.

Технологии

Современные веб-технологии: HTML5, CSS3, JavaScript и Python Flask на бэкенде. База данных - SQLite.

Ключевые возможности

Разграничение прав доступа

Разделение на роли: Студент и Преподаватель с различным уровнем доступа к функционалу.

Система заданий

Практические работы с пошаговым выполнением заданий и автоматической проверкой.

Мониторинг прогресса

Детальная статистика выполнения работ для студентов и преподавателей.
