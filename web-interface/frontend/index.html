О проекте

Учебный проект

Данный веб-интерфейс разработан в рамках выпускной квалификационной работы по специальности 10.05.02 «Информационная безопасность телекоммуникационных систем».

Цель

Создание удобной и функциональной платформы для анализа и обработки инцидентов информационной безопасности, предназначенной для студентов и преподавателей.

Технологии

Полигон построен на основе виртуальных машин (VirtualBox) и контейнеров (Docker). Интерфейс использует современные веб-технологии: HTML5, CSS3, JavaScript и Python (Backend).

Ключевые возможности

Разграничение прав доступа

Разделение на роли: Студент, Преподаватель и Администратор с различным уровнем доступа к функционалу.

Система заданий и сценариев

Доступ к библиотеке лабораторных работ и сценариев инцидентов ИБ для отработки практических навыков.

Автоматическая проверка

Система автоматически проверяет корректность выполнения заданий и отслеживает прогресс студентов.

Изолированная среда

Все эксперименты проводятся в безопасной, изолированной среде на базе виртуальных машин, не затрагивая основную инфраструктуру.

Примеры сценариев инцидентов

Компрометация веб-приложений

  • SQL-инъекции
  • Межсайтовый скриптинг (XSS)
  • Подбор учетных данных (Brute Force)

Несанкционированный доступ (НСД)

  • Анализ сетевого трафика
  • Эскалация привилегий
  • Обход механизмов аутентификации

Вредоносное ПО

  • Анализ поведения малвари
  • Расследование инцидентов
  • Методы обнаружения и противодействия
