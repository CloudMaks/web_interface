Рекомендации по использованию учебного полигона

Практические рекомендации для преподавателей и студентов

Введение

Данный учебный полигон создан для практического освоения навыков обработки инцидентов информационной безопасности. Платформа предоставляет интерактивную среду для отработки реальных сценариев ИБ в безопасных условиях.

Цель платформы: Сформировать у студентов практические навыки обнаружения, анализа и устранения инцидентов информационной безопасности.

Для студентов

1. Начало работы

Для доступа к платформе используйте демо-аккаунт или учетные данные, предоставленные преподавателем:

Логин: student
Пароль: student123

2. Структура практических работ

2.1. Подготовительный этап

  • Цель: Настройка системы мониторинга и уведомлений
  • Продолжительность: 30-40 минут
  • Особенности: Не оценивается баллами, но обязателен для выполнения

2.2. Практическая работа №1

  • Тема: Обработка инцидента несоответствующего использования ресурсов
  • Максимальный балл: 30 баллов
  • Количество заданий: 3 тестовых вопроса
  • Ключевые навыки: Анализ загрузки системы, работа с процессами, управление ПО

2.3. Практическая работа №2

  • Тема: Обработка инцидента несанкционированного доступа
  • Максимальный балл: 30 баллов
  • Количество заданий: 2 тестовых вопроса + 1 вопрос с вводом ответа
  • Ключевые навыки: Анализ журналов аутификации, настройка PAM, управление паролями

3. Рекомендации по выполнению

3.1. Подготовка к работе

  • Изучите теоретический материал перед началом практической работы
  • Убедитесь, что у вас есть доступ к виртуальной машине Kali Linux
  • Подготовьте блокнот для записи команд и наблюдений

3.2. Выполнение заданий

  • Внимательно читайте условия каждого задания
  • Следуйте пошаговым инструкциям в теоретической части
  • Используйте команды, приведенные в примерах
  • Проверяйте результаты выполнения команд

3.3. Работа с тестовыми заданиями

  • Внимательно читайте вопрос и все варианты ответов
  • Помните: за каждую лишнюю попытку снимается 1 балл
  • Максимальное количество попыток на задание - 10

4. Технические требования

4.1. Оборудование

  • Компьютер с доступом в интернет
  • Браузер: Chrome, Firefox, Edge или Safari последней версии
  • Разрешение экрана не менее 1280x720

4.2. Программное обеспечение

  • Виртуальная машина с Kali Linux (предоставляется преподавателем)
  • Доступ к терминалу Linux
  • Электронная почта для получения уведомлений (для подготовительного этапа)
